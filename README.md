# Web Diaries

**A Site where users can keep their biggest secrets safe.**

 Project Tech: 
- React (Client Side)
- Spring MVC (Server Side)
- Spring Security
- JWT Java Library
- Hibernate ORM
- Material UI 
- Java 8 features
- MySQL RDBMS
